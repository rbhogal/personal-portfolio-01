Game Save

About | Game Save   01

Web app where users can search, discover, and save information about video game titles pulled using the IGDB API (videogame database).

This was my first project using react and redux. My main aim for this project was to practice those skills, along with fetching data from an API and displaying the information, implementing (OAuth) authentication, some basic CRUD operations, and user accounts with data stored in firebase as my database and backend.
