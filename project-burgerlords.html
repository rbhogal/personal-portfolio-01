Burgerlords

A recreation of the LA restaurant Burgerlords' website. (Their original site has since changed)

My main aim with this project was to strengthen my HTML/Sass (SCSS), grid, flexbox, and javascript by trying to create a pixel-perfect copy of the original website. Learning keyframe animations and also making improvements where I saw fit.

Challenges

The biggest challenge was creating a custom hover effect that changes the background color using javascript.

Sounds weird. I know. I was unable to use CSS like normal due to how I had written the html, so I decided to add and remove helper classes using javascript to create a customized solution to the problem.

I think it was probably inexperience at the time and a CSS solution probably could work. Still what I was able to take away from this was first, to think ahead before writing out code to avoid these types of problems in the future, and secondly, that I am capable of creating my own solutions based on fundamental javascript knowledge if I need to.
